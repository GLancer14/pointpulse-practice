# Приложение "Мои задачи"

Стек технологий - React, Tanstack Query, React Router DOM 6, SASS, TypeScript.  
В проекте использована FSD-архитектура.  
  
Для запуска приложения - 
1. Клонируйте данный репозиторий командой `git clone https://github.com/GLancer14/pointpulse-practice`.
2. Откройте терминал в папке с клонированным репозиторием.
3. В терминале выполните команду `npm install`.
4. Переименуйте файл `.env.example` в корне проекта в `.env`.
5. Затем команду `npm start`.
6. Приложение доступно по адресу http://localhost:4173 (либо перейдите по ссылке, указанной в терминале).

## Бэкэнд приложения
  
Для запуска бэкэнда приложения - 
1. Откройте терминал в папке `/backend`.
2. В терминале выполните команду `npm install`.
3. Затем команду `npm start`.
